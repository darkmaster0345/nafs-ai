What Is This?

Nafs AI is an experimental simulation of a primitive conscious entity — an AI agent called Adam — who wakes up in an unknown world with zero prior knowledge. No language. No concepts. No culture. He discovers everything through experience alone: pain, hunger, warmth, fear, curiosity.

The Three Laws

Non-Intervention

2

Blank Slate

Adam knows nothing. He learns through pain and pleasure only.

The Cycle

Architecture

World Simulation (Python) ↓ Sensory Encoder → [ Adam's Brain (PPO + GRU) ] → Action ↑ ↓ Pain / Reward In-Life Memory ↑ ↓ Curiosity Bonus Dream Engine (SLEEP) ↓ Terminal + Web Dashboard (Thought / Dialogue / Action / Emotion) ↓ TensorBoard Dashboard

Features

🧠
PPO + GRU Brain
Proximal Policy Optimization with GRU recurrent layer. No pretrained models, no LLMs, no external APIs. True blank slate learning.
💀
Single-Life Mode
No episodes. Adam lives once. When he dies, the program ends. Run again to birth a new Adam in a new world.
🌍
10 Biomes
Desert, Forest, Tundra, Plains, Mountain, Swamp, Ocean, Jungle, Cave, Volcano. Procedurally generated 64x64 world map.
🌦️
8 Weather Types
Clear, Rain, Snow, Storm, Heatwave, Fog, Sandstorm, Blizzard. Dynamic Markov chain transitions affected by biome and time.
💭
6 Phases of Consciousness
Inner Voice, Vocabulary Discovery, Dialogue Gap, Persistent Memory, Fear/Pleasure Maps, Dual-Speed Processing.
🔍
Curiosity Module
Intrinsic motivation through count-based exploration. Novel states are rewarding. Curiosity naturally decays but never reaches zero.
💤
Dream Engine
Memory consolidation during SLEEP. Fear memories create nightmares, good memories create peaceful dreams. Exposure therapy for fears.
📊
Web Dashboard
Real-time web dashboard with vital signs, thought stream, biome map, action distribution, and more. Watch Adam live in your browser.

Six Phases of Consciousness

Phase 1: Inner Voice

Adam generates primitive thoughts from raw sensory experience. The thought IS the sensation, expressed in his limited vocabulary.

Phase 2: Vocabulary Discovery

Adam names things by combining existing vocabulary. "cold" + "pain" = "cold pain". Each discovered word carries the context of its birth.

Phase 3: Dialogue Gap (Subconscious)

Adam's outward expression is filtered through stress and emotion. The gap between thought and dialogue IS the subconscious.

Phase 4: Persistent Memory + Personality

Memory persists within one life. Fear triggers, good memories, and behavioral patterns accumulate, creating personality.

Phase 5: Fear/Pleasure Maps

Past fears and rewards are injected as signals into the observation vector. Adam learns from past fears, not just current danger.

Phase 6: Dual-Speed Processing

Fast PPO for immediate actions. Slow reflection every 20 ticks reviews experience and shapes future decisions.

Philosophy

Most AI projects ask "can AI pass as conscious?" from the outside. This project asks it from the inside. By denying Adam knowledge of his own nature, we create the only honest test: "Does it behave as if existence matters to it?"

"And He taught Adam the names of all things." — Quran 2:31

In this simulation — nobody teaches Adam the names. He finds them himself.

Created with curiosity. Observed with silence.
